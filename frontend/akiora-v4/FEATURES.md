# Akiora v4 - Функционал

Полное описание реализованного функционала приложения.

---

## 🔐 Аутентификация и Авторизация

### Вход в систему
- **Email аутентификация**: Вход через электронную почту с отправкой 6-значного кода подтверждения
- **OAuth провайдеры**: Интеграция с Discord и Yandex для быстрого входа
- **Многоязычная поддержка**: Интерфейс входа доступен на английском и русском
- **Защита**: Проверка статуса аутентификации и перенаправление незарегистрированных пользователей

### Логаут
- Выход со всех устройств
- Выход из текущей сессии
- Управление активными сессиями

---

## 👤 Профиль пользователя

### Основная информация
- **Никнейм**: Уникальное имя пользователя
- **Email**: Адрес электронной почты
- **Аватар**: 
  - Загрузка по URL
  - Загрузка файла (макс. 5MB)
  - Предпросмотр в режиме реального времени
- **Пол**: Выбор между "Мужской" и "Женский"
- **Дата рождения**: Запись дня рождения
- **Биография**: Свободный текст с описанием

### Социальные сети
- Поддерживаемые платформы:
  - VK, Telegram, Discord
  - YouTube, Twitch, SoundCloud
  - GitHub, Twitter, Instagram, Facebook
- Добавление/удаление социальных профилей
- Хранение ссылок и никнеймов для каждой платформы
- Цветовое кодирование платформ

### Настройки
- **Язык интерфейса**: Выбор между английским и русским
- **Уведомления**: Email-уведомления о турнирах и матчах
- **Тема**: Тёмный режим (светлый режим недоступен)

### Безопасность
- **Двухфакторная аутентификация**: Планируется (в разработке)
- **Управление сессиями**: Просмотр активных сессий, выход со всех устройств
- **Удаление аккаунта**: Опция удаления аккаунта (недоступна)

## 🏆 Турниры

### Список турниров
- **Фильтрация по статусу**:
  - Все турниры
  - Запланированные (Scheduled)
  - Активные/Идущие (Live)
  - Завершённые (Finished)
  - Отменённые (Canceled)

### Информация о турнире (карточка)
- **Статус**: Визуальный индикатор с цветовой кодировкой
- **Призовой фонд**: Отображение размера призового фонда
- **Время начала**: Форматированная дата и время
- **Формат**:
  - Размер команды (1v1, 2v2, 3v3, 5v5)
  - Тип сетки (Single Elimination, Double Elimination, Swiss, Round Robin)
  - Тип драфта (Classic, Fearless, Iron Man, All Random)
- **Количество участников**: Число команд в турнире
- **Статус регистрации**: Открыт ли турнир для новых участников

### Создание турнира
Форма создания с полями:
- **Название**: Призовой фонд турнира
- **Дата и время**: Когда начнётся турнир
- **Размер команды**: Выбор между 1v1, 2v2, 3v3, 5v5
- **Best Of**: Bo1, Bo3, Bo5
- **Тип сетки**: Single Elim, Double Elim, Swiss, Round Robin
- **Тип драфта**: Classic, Fearless, Iron Man, All Random
- **Открытая регистрация**: Может ли турнир присоединяться любой

## 🎮 Игровые серии (Game Series)

### Просмотр сетки турнира
- Иерархическая структура матчей
- Перемещение команд через раунды
- Отображение результатов матчей

### Раунды
- Просмотр активных раундов
- Информация о матчах в раунде
- Статус каждого матча

### Результаты
- История результатов матчей
- Финальные позиции команд
- Статистика турнира

### Черновой режим (Draft)
- Процесс драфта чемпионов
- Выбор персонажей командами
- Блокировка запрещённых персонажей
- Поддержка разных типов драфтов

## 💬 Мессенджер

### Список чатов
- **Типы чатов**:
  - Приватные (Private)
  - Публичные (Public)
- **Контексты чатов**:
  - Клубные чаты
  - Турнирные чаты
  - Чаты игровых серий
  - Системные уведомления
- **Фильтрация**: По типам и контексту

### Сообщения
- **Отправка**: Отправка текстовых сообщений
- **История**: Просмотр истории сообщений (последние 50)
- **Временные метки**: Время отправки каждого сообщения
- **Группировка**: Сообщения сгруппированы по датам
- **Автопрокрутка**: Автоматический скролл к последним сообщениям

### UI
- **Боковая панель**: Список всех чатов пользователя
- **Главная область**: Сообщения выбранного чата
- **Input область**: Ввод сообщения с кнопкой отправки
- **Клавиатурные комбинации**: Enter для отправки, Shift+Enter для переноса строки

## 🤝 Сообщество

### Поиск игроков
- **Поисковая строка**: Поиск по никнейму
- **Результаты**: Карточки игроков с информацией
- **Фильтрация**: Реальный поиск по запросу

### Информация об игроке
- **Никнейм**: Ник игрока
- **Email**: Адрес почты
- **Биография**: Описание профиля
- **Avatar**: Изображение профиля

### Клубы

#### Просмотр клубов
- **Список всех клубов**: Все клубы на платформе
- **Карточка клуба**:
  - Название клуба
  - Количество членов
  - Описание

#### Вступление в клуб
- Кнопка "Вступить" для авторизованных пользователей
- Требуется авторизация

#### Создание клуба
- **Название**: Имя клуба
- **Описание**: Опциональное описание
- Требуется авторизация

### Команды

#### Просмотр команд
- **Список всех команд**: Все командные группы
- **Карточка команды**:
  - Название команды
  - Тег команды (макс. 5 символов)
  - Количество членов

#### Подача заявки в команду
- Кнопка "Подать заявку" для авторизованных пользователей
- Требуется авторизация

#### Создание команды
- **Название**: Имя команды
- **Тег**: Уникальный 5-символьный тег (например: AKIRA)
- Требуется авторизация

### Группы

Функционал групп доступен для создания и управления малыми коллективами на платформе.

## 🎨 Дизайн и UI/UX

### Тёмная тема
- **Основные цвета**:
  - Фиолетовый (#a600ff) - основной акцент
  - Синий (#06B6D4) - альтернативный акцент

- **Кнопки**: Градиентные, с тенями свечения

- **Модалки**: Полноэкранные оверлеи с размытием фона

### Анимации
- **Появление**: Rise-up эффект при загрузке
- **Переходы**: Плавные анимации между состояниями
- **Hover эффекты**: Интерактивная обратная связь

## 🔧 Технический стек

### Фронтенд
- **Framework**: React 19 с TanStack Router
- **Стилизация**: Tailwind CSS
- **Иконки**: React Icons, Lucide React
- **Состояние**: TanStack Query для серверного состояния
- **Интернационализация**: Paraglide JS
- **Типизация**: TypeScript

### Backend интеграция
- **API**: Собственный API через lib/api.ts
- **Аутентификация**: OAuth + Email code authentication
- **WebSocket**: Для мессенджера (планируется)
